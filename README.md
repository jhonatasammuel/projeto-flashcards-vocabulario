# EasyLearning: Website de revisão de vocabulário com Flashcards
## 📋 Sobre o Projeto
O EasyLearning é um projeto de extensão universitária desenvolvido para o Centro Universitário Internacional UNINTER (Escola Superior Politécnica). Trata-se de uma plataforma educacional focada na democratização do aprendizado de idiomas, utilizando metodologias de memorização ativa para estudantes de diversos perfis.

## 🎯 Setor de Aplicação e Impacto Social
O foco principal é o setor educacional, especificamente escolas públicas e comunidades de baixa renda. O projeto visa:

*  Inclusão Digital: Ferramenta gratuita adaptada para diferentes níveis de alfabetização.
  
* Acessibilidade: Design pensado para usuários com deficiências visuais ou motoras, oferecendo navegação por teclado e compatibilidade com leitores de tela.
  
* Democratização: Redução de barreiras tecnológicas através de uma interface leve e de baixo custo operacional.

## 🌍 Objetivos de Desenvolvimento Sustentável (ODS)
Este projeto está alinhado com as metas da Agenda 2030 da ONU, atendendo diretamente ao objetivo:

* ODS 04: Educação de Qualidade – Assegurar a educação inclusiva e equitativa e de qualidade, e promover oportunidades de aprendizagem ao longo da vida para todos.

## ⚙️ Objetivos Técnicos e Funcionais
* Metodologia de Aprendizado: Implementação de Flashcards Interativos baseados na técnica de Repetição Espaçada para otimizar a memorização de vocabulário, transcrições fonéticas e exemplos de uso.

* Desenho Universal e Acessibilidade: * Interface responsiva (Desktop e Mobile).

* Suporte a reconhecimento de voz para entrada de dados.

* Recursos de leitura em voz alta para suporte a pessoas com baixa visão.

* Autonomia do Usuário: Painel de indicadores de progresso (dias consecutivos, taxa de acerto e evolução de vocabulário) para estimular o estudo independente.

## 🧪 Metodologia e Planejamento
O desenvolvimento do projeto segue uma estrutura rigorosa de levantamento de requisitos e análise de viabilidade técnica, focando na experiência do usuário (UX) em contextos de conectividade limitada.

## 🚀 Resultados Esperados
* Aumento na retenção de vocabulário e expressões idiomáticas por parte dos alunos.

* Engajamento de estudantes de escolas públicas através de uma plataforma intuitiva.

* Estímulo à autonomia no processo de aprendizagem, mensurado por registros de frequência e desempenho dentro do sistema.

## 📝 Considerações de Engenharia
O planejamento deste software permitiu o desenvolvimento de empatia com o usuário final, garantindo que as funcionalidades atendam às necessidades reais do público. Os principais desafios de engenharia abordados incluem a gestão de tempo no ciclo de desenvolvimento e a estruturação de uma arquitetura que suporte inclusão digital sem necessidade de hardware especializado.
